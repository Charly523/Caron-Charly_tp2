## Mon choix du programme
J'ai choisi ce programme pour trois grandes raisons:
* La programmation
* Mon intérêt pour les jeux vidéo
* Et tous ce qui touche à la création web
## Mon profil multimédia
* Créatif: mon côté créatif n'est pas très développé​
* Technique: je suis confortable avec mes travaux car j'ai toujours été plus technique
## Mes forces
Mes plus grandes forces pour ce domaine sont:
* Ma grande patience 
* Mon organisation, je ne peux pas travailler si ce n'est pas organiser

Mes plus grandes faiblesses dans ce domaine sont:
* Ma timidité qui m'empêche de parler aux autres
* Je ne pose pas de questions
